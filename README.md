GroupPay

GroupPay is a modern, user-friendly platform to manage group finances and shared expenses. It allows users to create wallets, track transactions, manage members, and maintain transparent group payments.
This project is built with Java (Spring Boot) for the backend and HTML, CSS, JavaScript, TailwindCSS for the frontend.
